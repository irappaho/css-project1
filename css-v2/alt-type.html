Alternative Typography

Some alternative type options for use on the web, all available for free via Google Fonts.

Rubik is a sans-serif font family with slightly rounded corners. It's available in five weights and would work well for headings.

Roboto is a neo-grotesque sans-serif typeface family developed by Google. We propose using it for body text.

Libre Baskerville is a serif typeface based on the American Type Founder's Baskerville from 1941, but it has a taller x-height, wider counters and a little less contrast, that allow it to work well for reading on-screen. We propose using it for blockquotes/pull-quotes.
